Microservicio de Notificaciones Firebase

Solución empresarial completa para el envío masivo de notificaciones push a través de Firebase Cloud Messaging. Diseñado para alta disponibilidad, escalabilidad y fácil integración con sistemas Oracle APEX y otros sistemas.

Ver Dashboard Documentación

Características Principales

Todo lo que necesitas para gestionar notificaciones a escala empresarial

Alto Rendimiento

Procesamiento en cola con Redis y Bull para manejar miles de notificaciones simultáneamente con tiempos de respuesta óptimos.

Análiticas en Tiempo Real

Dashboard completo con estadísticas en vivo, métricas de entrega y análisis de rendimiento con WebSockets.

Seguridad Empresarial

Autenticación por API Keys, rate limiting, validación de datos y logs completos para auditoría y cumplimiento.

Fácil Integración

REST API completa con documentación Swagger, webhooks para Oracle APEX y SDKs para diferentes plataformas.

Configuración Flexible

Sistema de configuración dinámico, soporte para múltiples tipos de notificaciones y personalización completa.

Arquitectura Robusta

Diseño modular con SQLite para persistencia, manejo de errores robusto y recuperación automática.
